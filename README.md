# AuthTrail: CLI Discipline Tracker

**AuthTrail** is a terminal-based productivity tool designed to enforce discipline through structured command logging. It forces the user to consciously "Approve" or "Reject" daily habits and tracks performance in a structured JSON database.

- **Logic Constraints:** Enforces specific proofs (e.g., "Must upload photo") before approving tasks.
- **Portable:** compiled into a standalone `.exe` for Windows.
